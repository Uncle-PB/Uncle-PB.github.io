    Elasticsearch-Head 插件使用

    通过elasticsearch-Head 插件访问Es ，以及一些常用语句的记录

    安装插件

    因为谷歌商店不能用，所以通过Gitee下载。地址：elasticsearch-Head插件
    如果直接拖拽到Chrome会不能使用，告诉你非Chrome来源的
    文件后缀名”.crx”改为“.rar”
    解压到某个文件夹里
    Chrome“加载已解压的扩展程序”按钮加入文件夹就可以使用了。

    常用语句

    # 创建有类型、映射的索引

    number_of_shards：分片数量（按机器节点数量计算，创建后不能更改）
    number_of_replicas：副本数量

    ES-Head 建立索引 

    PUT http://127.0.0.1:9200/lrs_index
{
  "settings":{
      "number_of_shards":3,
      "number_of_replicas": 1
    },
  "mappings": {
    "properties": {
      "lrs_field": {
        "type": "text",
        "analyzer": "ik_max_word",
        "search_analyzer": "ik_max_word"
      }
    }
  }
}

    # 删除索引

    ES-Head 删除索引 

    DELETE http://127.0.0.1:9200/lrs_index

    # 获取索引结构

    ES-Head 获取索引结构 

    GET http://127.0.0.1:9200/lrs_index/_mapping

    # 插入数据

    1，指定ID PUT/POST 都行，存在则更新
    2，不指定ID，自动生成Id 只能POST
    3，POST的时候自动生成mapping 如果POST的参数不是严格的已经定义好的，会根据POST的情况自动进行创建对应的mapping；
    4，ES插件执行需要加 /_doc

    ES-Head 插入数据 

    PUT or POST http://127.0.0.1:9200/lrs_index/1
{
  "lrs_field":"xxx"
}

    # 查看分词情况

    形式：/${index}/${type}/${id}/_termvectors?fields=${fields_name}
    例子：lrs_index/_doc/1/_termvectors?fields=lrs_field

    ES-Head 查看分词情况 

    GET lrs_index/1/_termvectors?fields=lrs_field

    # 使用分词器分析

    ES-Head 使用分词器分析 

    POST lrs_index/_analyze 
{
  "analyzer": "ik_smart",
  "text": "_2_3_22_12_"
}

    # 查询

    ES-Head 查询 

    POST lrs_index/_doc/_search
根据条件查询
{
  "query": {
    "match": {
      "_id": "22"
    }
  }
}
查询全部
{
  "query": {
    "match_all": {}
  }
}

    # 条件\范围查询

    POST xxxx{-索引-}/_search
{
  "size": 1000{-查询数量-},
  "query": {
    "bool": {
      "must": [
        {
          "range": {
            "date{-字段名称-}": {
              "gte": "2021-03-11",
              "lte": "2021-03-18"
            }
          }{-等同mysql date between "2021-03-11" and "2021-03-18" -}
        },
        {
          "bool": {
            "must": [
              {
                "term": {
                  "city_id": "0"
                }
              },{-等同mysql city_id = 0 -}
              {
                "terms": {
                  "type_id": [
                    "1",
                    "2"
                  ]
                }{-等同mysql type_id in (1,2) -}
              }
            ]
          }
        }
      ]
    }
  }
}

    # 根据查询删除

    ES-Head 根据查询删除 

    POST lrs_index/_doc/_delete_by_query
{
  "query": {
    "match": {
      "_id": "22"
    }
  }
}
